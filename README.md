# Oh-My

OpenCode 자동화 프로젝트

## 기능

- GitHub Issues에서 `/oc` 코멘트로 자동 코딩 작업 실행
- Self-hosted Runner로 집 PC에서 실행
- 자동 PR 생성 및 리포트 작성
- Gemini AI 모델 사용

## 사용 방법

1. 이슈 생성
2. 코멘트에 `/oc [작업 내용]` 작성
3. 자동으로 PR 생성됨

## 설정 가이드

상세 설정 방법은 다음 파일 참고:
- `QUICK_START.md` - 빠른 시작
- `SETUP_GUIDE.md` - 상세 가이드
- `STEP_BY_STEP.md` - 단계별 가이드
